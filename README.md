# 🎓 Student Study Pattern vs Performance Analyzer

## 📌 Overview
This project analyzes how student habits such as study hours, sleep, mobile usage, and attendance affect academic performance using basic Machine Learning techniques.

---

## 🎯 Problem Statement
Students often do not realize how their daily habits influence their marks. This project aims to identify key factors affecting student performance and predict marks based on these factors.

---

## 📊 Dataset
Data was collected using Google Forms from students. The dataset includes:
- Study Hours per Day
- Sleep Hours per Day
- Daily Mobile Usage
- Attendance Percentage
- Internal Marks

---

## ⚙️ Methodology
1. Data Collection using Google Forms  
2. Data Cleaning and Preprocessing  
3. Data Visualization using graphs  
4. Machine Learning Models:
   - Linear Regression
   - Decision Tree Regressor  

---

## 📈 Results & Insights
- 📚 Study hours have a strong positive impact on marks  
- 🏫 Attendance significantly improves performance  
- 📱 High mobile usage negatively affects marks  
- 😴 Sleep has moderate influence
