# 📊 Dataset Information

This folder contains the dataset used for the project.

## 📌 Source
The data was collected using Google Forms from students.

## 📋 Features in Dataset
- StudyHours → Number of hours studied per day  
- SleepHours → Daily sleep duration  
- MobileUsage → Time spent on mobile per day  
- Attendance → Attendance percentage  
- Marks → Internal marks obtained  

## ⚠️ Note
- Data is cleaned and converted to numerical format before analysis.
- Timestamp column (if present) is removed during preprocessing.
